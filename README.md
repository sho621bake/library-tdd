# 図書貸出システム（TDDハンズオン成果物）

## 実行手順

npm install
npm test

## 仕様概要

- 蔵書（Book）: ISBN・タイトル・著者で管理。貸出中/貸出可能の状態を持つ
- 会員（Member）: ID・名前で管理。借りている本のリストを保持
- 貸出記録（Loan）: 貸出日・返却期限（14日後）。延滞判定を提供
- 延滞料金（Fine）: 1日50円（切り上げ）。値オブジェクトとして不変
- 貸出ルール: Strategy パターンで交換可能（上限3冊、延滞中NG、貸出不可チェック）
- 通知: Notifier インターフェースで外部依存を切り離し（DI）

## 設計判断

- Library はコーディネーター。ドメインロジックは各クラスに分散
- LendingRule は順序非依存。ルール追加時は新クラスを implements するだけ
- Notifier は Optional。失敗しても貸出/返却は成功する
- 不変条件: Book の貸出状態と activeLoans は常に整合
